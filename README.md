# Discord bot : Pokebot

## About
It's a discord bot I use with my friends to catch and trade pokemon

## Tech Stack
- Discord JS

## Getting Started

### Prerequisites
- Node.js
- npm package manager
- A discord bot

### Installation

Clone the repository :
```
git clone https://github.com/AS-Ven/Pokebot
```

Install the dependencies :
```
npm install
```

Add a .env file :
```
BOT_TOKEN=""
```

Start the project :
```
node .
```
